# CoupleFriendly 💑

A private and secure chat app built exclusively for couples.

## About

CoupleFriendly is an Android app that gives couples their own private space to chat. It features a secure login system with an extra passcode layer, ensuring only the right people can access the conversation.

## Features

- Email and password authentication
- Passcode screen for added security after login
- Private one-on-one chat
- Clean and modern UI built with Jetpack Compose

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material3
- **Architecture:** Clean Architecture (auth, data, navigation)
- **Minimum SDK:** API 26

## Getting Started

1. Clone the repository
2. Open in Android Studio
3. Build and run on an emulator or physical device

## Project Structure

```
app/src/main/java/com/example/couplefriendly/
├── auth/          # Login, OTP, and passcode screens
├── data/          # Data layer and repositories
└── navigation/    # Navigation graph and routing
```
